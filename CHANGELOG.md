# Changelog

All notable changes to Creator Asset Studio releases will be documented here.

## [0.1.0] - 2026-03-12

### Added

- Initial public Windows installer release
- Export workflow for YouTube channels, playlists, and single videos
- Audio, video, thumbnail, description, metadata, subtitle, transcript, and chapter export options
- Manifest generation and RSS feed builder
- Metadata editing, retry support, and dependency visibility
- Branded installer, application icon, and release screenshots

### Notes

- Source code is maintained in a private repository
- Public downloads are intended to be distributed through GitHub Releases
